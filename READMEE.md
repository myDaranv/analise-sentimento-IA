# Análise de Sentimento

Este projeto é um exemplo simples de análise de sentimento usando Python.

## O que o programa faz
Analisa uma frase e identifica se o sentimento é positivo, negativo ou neutro,
com base em palavras pré-definidas.

## Objetivo
Projeto criado para fins de aprendizado e portfólio como estudante de Ciência da Computação.

## Tecnologias usadas
- Python (sem bibliotecas externas)
